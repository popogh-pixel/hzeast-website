# 产品图片和视频上传指南

## 📁 文件存放位置

所有产品媒体文件统一存放在：`uploads/` 文件夹

```
website-v2/
├── uploads/
│   ├── banner1.jpg          # Hero Banner 1
│   ├── banner2.jpg          # Hero Banner 2
│   ├── product-xxx.jpg      # 产品图片
│   ├── product-xxx.mp4      # 产品视频
│   └── about-video.mp4      # 公司介绍视频
```

---

## 🖼️ 产品图片上传

### 1. 准备图片
- **格式**: JPG, PNG, WebP
- **推荐尺寸**: 1200x800px 或 800x1200px（横图或竖图）
- **文件大小**: < 500KB（优化加载速度）
- **命名**: `product-品类 - 编号.jpg`（如：`product-dress-001.jpg`）

### 2. 上传步骤
```bash
# 方法 1: 直接复制到 uploads 目录
cp ~/Desktop/product-photo.jpg ~/.openclaw/workspace/website-v2/uploads/

# 方法 2: 使用 Finder
# 打开 ~/.openclaw/workspace/website-v2/uploads/ 拖入图片
```

### 3. 在页面中展示

**产品展示页** (`pages/products.html`):
```html
<div class="product-card">
    <div class="product-image">
        <img src="../uploads/product-dress-001.jpg" alt="连衣裙">
    </div>
    <h3>时尚连衣裙</h3>
    <p>描述文字...</p>
</div>
```

## 🎥 产品视频上传

### 1. 视频规格
- **格式**: MP4 (H.264 编码)
- **分辨率**: 1920x1080 (1080p) 或 1280x720 (720p)
- **时长**: 15-60 秒（产品展示）
- **文件大小**: < 50MB
- **命名**: `product-品类 -video.mp4`

### 2. 上传步骤
```bash
cp ~/Desktop/product-video.mp4 ~/.openclaw/workspace/website-v2/uploads/
```

### 3. 在页面中展示

**方案 A: 自动播放背景视频**
```html
<div class="product-video-container">
    <video autoplay muted loop playsinline class="product-video">
        <source src="../uploads/product-dress-video.mp4" type="video/mp4">
        您的浏览器不支持视频标签
    </video>
</div>
```

**方案 B: 带播放控件的视频**
```html
<div class="product-video-card">
    <video controls poster="../uploads/product-video-cover.jpg">
        <source src="../uploads/product-dress-video.mp4" type="video/mp4">
    </video>
    <h3>产品展示视频</h3>
</div>
```

## 🏭 公司介绍视频

### 1. 视频规格
- **格式**: MP4 (H.264 编码)
- **分辨率**: 1920x1080 (1080p)
- **时长**: 1-3 分钟
- **文件大小**: < 100MB
- **命名**: `about-video.mp4` 或 `company-intro.mp4`

### 2. 上传步骤
```bash
cp ~/Desktop/company-intro.mp4 ~/.openclaw/workspace/website-v2/uploads/
```

### 3. 在"关于我们"页面展示

**pages/about.html**:
```html
<section class="about-video-section">
    <div class="container">
        <h2>走进伊斯特</h2>
        <div class="video-wrapper">
            <video controls poster="../uploads/about-video-cover.jpg">
                <source src="../uploads/about-video.mp4" type="video/mp4">
            </video>
        </div>
        <p>33 年专业服装制造历程...</p>
    </div>
</section>
```

## 📊 优化建议

### 图片优化
```bash
# 使用 ImageOptim 或在线工具压缩图片
# 推荐工具：
# - TinyPNG (https://tinypng.com/)
# - ImageOptim (Mac 应用)
# - Squoosh (https://squoosh.app/)
```

### 视频优化
```bash
# 使用 HandBrake 压缩视频
# 推荐设置:
# - 编码器：H.264
# - 质量：RF 22-25
# - 音频：AAC 128kbps
```

### 懒加载（提升性能）
```html
<img src="product.jpg" alt="产品" loading="lazy">
```
